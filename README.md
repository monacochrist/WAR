## Download & Run

Download the latest AppImage from the [Releases page](https://github.com/monacochrist/WAR/releases/latest):

```
wget https://github.com/monacochrist/WAR/releases/latest/download/WAR-x86_64.AppImage
chmod +x WAR-x86_64.AppImage
./WAR-x86_64.AppImage
```

**Requirements:** Pipewire, Wayland, and Freetype — present on any modern Linux desktop.

Full keybindings and controls: **[CONTROLS.md](CONTROLS.md)**
